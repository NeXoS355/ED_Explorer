EDExplorer helps you by exploring the Uniserve in Elite Dangerous more efficiently.

- get the full System Overview after you detected all Planets
- decide at a glance if it is worth mapping a planet with the DSS
- get detailed Planet Information
  - Materials
  - Planet Signals
  - Distance from central Star
  - estimated Value of Discovery


Build with Python because i need it to run on Linux and i happen to know Python a little bit.
Requirements
- textual
- rich

EDExplorer reads the following file from Elite Dangerous
- Journal*.json
- NavRoute.json
- Status.json
